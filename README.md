### 👨‍💻 What I'm up to
- 🔭 I’m currently working on a **5-stage pipelined RISC-V CPU core (RV32IFD)** using Chisel.
- ⚙️ I have hands-on experience in building an **ARM Cortex-M0 MCU** with a Harvard banked memory architecture.
- 🌱 I’m constantly exploring advanced topics in **Cell-based IC Design Flow**, **FPGA Prototyping**, and **Multi-MCU System Integration**.

### 🛠️ Tech Stack
- **Hardware/RTL:** Verilog, SystemVerilog, Chisel
- **Software/Firmware:** C/C++, Python, ARMv6 / RV32IFD Assembly
- **MCU & Protocols:** STM32, ESP32, 8051, UART, I²C, SPI, MQTT
- **EDA Tools:** Quartus, Vivado, ModelSim, VCS, nWave, Innovus, Calibre
